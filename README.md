## 项目介绍

Cursor Manager 是一款专为 [Cursor](https://cursor.sh/) AI编辑器设计的增强管理工具。它可以帮助您解锁 Cursor 的全部功能，管理浏览器会话，重置机器ID，并提供更流畅的使用体验。无论您是免费用户还是付费用户，Cursor Manager 都能帮助您充分利用 Cursor 编辑器的强大功能。

[![项目演示](https://i.imgur.com/PLACEHOLDER2.png)](https://youtu.be/PLACEHOLDER)

## 主要功能

✅ **重置机器ID** - 解决设备限制问题  
✅ **完全重置Cursor** - 清除所有配置和缓存  
✅ **语言切换** - 支持中英文界面  
✅ **配置查看** - 显示当前Cursor配置信息  
✅ **账户授权检查** - 验证当前账户状态  
✅ **令牌限制绕过** - 优化令牌使用策略  
✅ **浏览器管理** - 自动管理浏览器会话  
✅ **无痕浏览模式** - 保护隐私的浏览方式  
✅ **邮箱验证码获取** - 简化账号验证流程  
✅ **Cursor进程管理** - 一键关闭应用或进程  
✅ **自动更新** - 一键更新到最新版本  

## 下载安装

### 系统要求

- Windows 10/11 操作系统
- 已安装 Cursor 编辑器

### 安装方法

1. 从 [GitHub Releases](https://github.com/Xie-Rutai/cursor-manager/releases/latest) 下载最新版本
2. 解压缩下载的文件到任意位置
3. 运行 `Cursor-Manager.exe` 启动程序

## 使用说明

### 基本操作

1. 启动 Cursor Manager
2. 从主菜单选择所需功能
3. 按照屏幕提示完成操作

### 功能详解

#### 1. 重置机器ID

重置 Cursor 的机器标识，适用于解决多设备使用限制问题。

```
操作步骤:
1. 从主菜单选择 "1. 重置机器ID"
2. 确认操作
3. 等待重置完成
```

#### 2. 完全重置Cursor

清除所有 Cursor 配置和缓存，恢复到全新安装状态。

```
操作步骤:
1. 从主菜单选择 "2. 完全重置Cursor"
2. 确认操作 (注意: 此操作将删除所有设置)
3. 等待重置完成
```

#### 3. 语言切换

切换 Cursor Manager 的界面语言。

```
操作步骤:
1. 从主菜单选择 "3. 切换语言"
2. 选择所需语言
```

#### 4. 显示配置信息

查看当前 Cursor 的配置详情。

```
操作步骤:
1. 从主菜单选择 "4. 显示配置信息"
2. 查看配置详情
```

#### 5. 检查账户授权

验证当前 Cursor 账户的授权状态。

```
操作步骤:
1. 从主菜单选择 "5. 检查账户授权"
2. 查看授权状态
```

#### 6. 绕过令牌限制

优化 Cursor 的令牌使用策略，提升AI功能使用体验。

```
操作步骤:
1. 从主菜单选择 "6. 绕过令牌限制"
2. 按照提示完成设置
```

#### 7. 浏览器管理器

管理专用浏览器会话，简化 Cursor 登录和使用流程。

```
操作步骤:
1. 从主菜单选择 "7. 浏览器管理器"
2. 选择所需的浏览器操作
```

#### 8. 无痕浏览模式

启动隐私保护的浏览模式。

```
操作步骤:
1. 从主菜单选择 "8. 无痕浏览模式"
2. 等待浏览器启动
```

#### 9. 邮箱验证码获取

简化 Cursor 账号注册和登录的验证流程。

```
操作步骤:
1. 从主菜单选择 "9. 邮箱验证码获取"
2. 按照提示操作
```

#### 10-11. Cursor应用管理

快速关闭 Cursor 应用或终止相关进程。

```
操作步骤:
- 选择 "10. 关闭Cursor应用" 优雅关闭应用
- 选择 "11. 关闭Cursor进程" 强制终止进程
```

#### 12. 检查软件更新

自动检查并更新 Cursor Manager 到最新版本。

```
操作步骤:
1. 从主菜单选择 "12. 检查软件更新"
2. 如有更新，按照提示完成安装
```

### 项目结构

```
cursor-manager/
├── main.py               # 程序主入口
├── updater.py            # 自动更新模块
├── common_utils.py       # 通用工具函数
├── reset_machine.py      # 机器ID重置实现
├── browser_manager.py    # 浏览器管理实现
├── locales/              # 多语言支持文件
│   ├── en.json           # 英文翻译
│   └── zh.json           # 中文翻译
├── resources/            # 资源文件
│   └── icons/            # 图标文件
└── dist/                 # 编译后的可执行文件
```

## 常见问题

### Q: 软件安全吗？
**A:** Cursor Manager 是开源软件，您可以检查源代码确保安全性。本工具仅对 Cursor 编辑器的配置文件和环境进行管理，不会收集任何个人信息。

### Q: 更新失败怎么办？
**A:** 如果自动更新失败，请尝试：
1. 检查网络连接（GitHub 在某些地区可能需要代理）
2. 手动下载最新版本替换旧文件
3. 确保您有足够的权限修改程序文件

### Q: 为什么有些功能需要管理员权限？
**A:** 某些功能（如重置机器ID）需要修改系统受保护的文件或注册表，因此需要管理员权限才能正常工作。

### Q: 如何正确卸载？
**A:** Cursor Manager 不会在系统中安装额外文件，直接删除程序文件夹即可完成卸载。
